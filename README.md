

# Demo 项目

这是一个基于 Spring Boot 的简单示例项目。

## 功能

- 提供一个 /hello 接口
- 支持传入 name 参数，返回问候语

## 安装和运行

1. 克隆项目到本地：

   ```bash
   git clone https://github.com/你的用户名/你的项目.git
   cd 你的项目
   ```

2. 使用 Maven 打包：

   ```bash
   mvn clean package
   ```

3. 运行项目：

   ```bash
   java -jar target/demo-0.0.1-SNAPSHOT.jar
   ```

4. 访问接口：

   打开浏览器访问：http://localhost:8080/hello?name=你的名字
